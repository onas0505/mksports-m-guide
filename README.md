# mksports-m-guide

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面内容涵盖体育相关资讯、指南或信息整理，但不针对任何特定域名或网站。仓库的主要目的是作为静态页面的集中存储与展示空间，便于浏览与引用。

## 目录结构

```
mksports-m-guide/
├── index.html          # 主入口页面（如有）
├── pages/              # 独立 HTML 页面存放目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 共用资源文件（样式、图片、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放所有独立 HTML 页面，每个页面对应一个独立主题或内容。
- `assets/`：存放页面共用的样式、脚本及图片资源，保持页面间的统一性。

## 页面归档说明

- 所有 HTML 页面均为独立文件，可直接在浏览器中打开查看。
- 页面不依赖外部服务器或数据库，仅使用本地资源。
- 归档内容不针对任何特定网站或平台，仅为信息汇总与展示。
- 页面内容可能随时间更新或补充，但不会频繁变动。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面会通过 commit 记录。
- 欢迎通过 Issue 提出建议或报告问题，但请注意保持内容的中立与通用性。
- 不接受直接提交 Pull Request 修改内容，如有需要可先讨论。
- 所有页面遵循基本的前端开发规范，尽量保证在不同浏览器下的兼容性。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `index.html` 或 `pages/` 下的任一 HTML 文件即可浏览。
3. 也可通过 GitHub Pages 等静态托管服务部署后在线访问（需自行配置）。
